# Overview

Mana Agent is a repository assistant for exploring and understanding codebases. It provides tooling for searching source code, inspecting structure, reading files, and generating evidence-backed answers about a project.

## What it helps with

- Discovering project structure and important files
- Searching for exact text or conceptual code locations
- Reading source files and AST-derived symbols
- Verifying behavior with project checks and tests

## How it is intended to be used

The assistant is designed to gather repository evidence before making conclusions. It prefers direct inspection of the codebase, file contents, and automated checks over guesswork.

## Repository areas

This project includes source code under `src/mana_agent`, tests under `tests`, and documentation under `docs`.
